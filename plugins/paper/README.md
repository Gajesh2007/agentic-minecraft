# Agentic Survival — Paper MC Plugin

Java plugin for Paper 1.21.4. Handles server-side game mechanics:

- Alliance system (form/break, shame broadcast)
- Territory claims (intrusion alerts)
- Combat state machine (challenge/accept/decline, turn-based)
- Death elimination (permanent, spectator mode)
- Budget-on-kill webhook bridge
- World border phased shrinking
- Custom scoreboard
- Trade logging
- Event scheduler (supply drops, storms)

Phase 4 — not yet implemented.
